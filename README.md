🩺 Doctor Portfolio Website
A professional, responsive portfolio website designed for doctors to showcase their services, expertise, and allow patients to book appointments online.

🚀 Features
Professional Landing Page: Clean and modern design tailored for medical professionals.

Appointment Booking: Includes a dedicated book.html page for patient inquiries.

Fully Responsive: Optimized for viewing on desktops, tablets, and mobile phones.

Image Gallery: Organized assets within the image folder for easy management.

🛠 Tech Stack
HTML5: For page structure and semantic markup.

CSS3: For styling, layout, and responsive design.

GitHub Pages: Hosted and ready for live preview.
